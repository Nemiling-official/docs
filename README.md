# Документация Nemiling

Официальная документация **Nemiling** — платформы для монетизации Telegram-каналов и групп через подписки, платный контент, консультации и автоматизацию продаж. Работаем с 2019 года.

Инструмент Nemiling — бот [@Nemilin_bot](https://t.me/Nemilin_bot). Помогаем авторам, экспертам и сообществам зарабатывать на своих Telegram-проектах. Деньги поступают напрямую авторам, без посредников.
